Introduction

This is a small test server. Contains a series of "end_points" serving json objects. Designed to study front-end frameworks such as 'vuejs', 'reactjs', etc. The idea is to replace the "hour and a half" search on the internet for free "apis" when studying the frameworks mentioned above.
